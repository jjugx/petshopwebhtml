วิธีการชำระเงิน

1. บัตรเครดิต/บัตรเดบิต

- ชำระเงินด้วยบัตรเครดิต/เดบิต ที่มีสัญลักษณ์ Visa, Mastercard หรือ JCB
- ใส่หมายเลขบัตร 16 หลัก
- ใส่ชื่อตามบัตรเครดิต/เดบิต
- กรอกวันหมดอายุ
- ใส่รหัส เลขท้าย 3 ตัวบนหลังบัตร (CVV)

2. ATM

- เลือกธนาคารที่ต้องการชำระเงิน
- ลูกค้าสามารถชำระเงินด้วยการโอนเงินผ่านเคาน์เตอร์ธนาคาร ATM ภายใน 48 ชม.

3. Internet Bankink

- เลือกธนาคารที่ต้องการชำระเงิน
- ลูกค้าสามารถชำระเงินด้วยการสแกน QR code ผ่านธนาคาร

4. ชำระเงินปลายทาง

ลูกค้าสามารถชำระเงินกับพนักงานจัดส่งของ เมื่อได้รับสินค้า โดยมีค่าส่งตามน้ำหนักและขนาดที่กำหนด
